# Exention Finder

Расширение мониторит заданную страницу, ищет совпадения со списком слов, подсвечивает их ярким фоном и подает звуковой сигнал.

## Возможности

- Постоянный мониторинг DOM-обновлений (MutationObserver + периодический скан).
- Подсветка найденных слов и звуковое уведомление.
- Список слов хранится локально в `chrome.storage.local` и сохраняется после обновлений страницы.
- Добавление/удаление отдельных слов.
- Импорт/экспорт списка слов (JSON или текстовый файл).

## Как использовать

1. Установите расширение в режиме разработчика.
2. Откройте страницу, которую нужно мониторить.
3. В попапе нажмите «Использовать текущую вкладку» или укажите часть URL вручную.
4. Добавьте слова в список.

## Важное про фоновый мониторинг

Контент-скрипт работает даже если вкладка не активна, но браузер может ограничивать частоту таймеров и скриптов в фоновых вкладках. Поэтому мониторинг продолжится, однако обновления могут обрабатываться с небольшой задержкой в зависимости от политики браузера.
